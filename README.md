Haftalık Rapor Denetiminde kullanılabilecek basit UI tasarımlı bir C# .NETFrameWork çalışması.
Öncelikle kendi divizyonunuzda bulunan şahısları personel kayıt ile ekliyorsunuz. Bütün personellerin tek bir dosyada olması tavsiye edilir.
Daha sonradan haftalık rapor denetiminde Personel Yükle butonuna basarak personel.txt dosyalarını aktarıyoruz. 
Liste kaydet butonu ile personel listesini kaybediyoruz ve uygulamadan çıkıp girseniz bile liste kalıyor.(Listeyi güncellemek için temizle veya sil yapabilirsiniz. Yaptıktan sonra tekrar liste kaydet butonuna tıklayınız).
Divizyon seçimi ile raporunu doldurmak istediğiniz divizyonu seçiyorsunuz, Sonradan listedeki Personelleri seçip personel seç butonuna basıyoruz. Sonradan personellere ait raporlandırmaları giriyoruz ve saat hesapla butonuna basıyor en son ise saat aktar butonuna tıklıyor ve işlem sonunda çıktı butonuna basarak HTML kodnu alıyoruz.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
A simple UI design C# .NETFramework project that can be used in Weekly Report Audits.
First, add the individuals in your division using the personnel registration. It is recommended that all personnel be in a single file.
Later, in the weekly report audit, we import the personnel.txt files by clicking the Load Personnel button. 
We save the personnel list with the Save List button, and the list remains even if you exit and re-enter the application. (You can clear or delete the list to update it. After doing so, click the Save List button again).
You select the division you want to fill out the report for with the division selection, then select the personnel from the list and click the Select Personnel button. Then, we enter the reports for the staff members and click the “Calculate Hours” button. Finally, we click the “Transfer Hours” button and, at the end of the process, click the “Output” button to get the HTML code.
